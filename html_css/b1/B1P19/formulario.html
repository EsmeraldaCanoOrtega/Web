Parte 1

  1. Color: nos muestra un color picker, el valor elegido se mostrará en formato RGB.
  2. Date: este campo nos muestra un calendario para seleccionar una fecha
  3. Datetime: en este campo se puede guardar la fecha y la hora en formato texto
  4. Datetime-local: este campo nos muestra un calendario para seleccionar una fecha y un reloj para seleccionar la hora
  5. Email: este campo permite introducir texto, que no será validado si no es un formato de email valido
  6. File: este campo permite subir un archivo
  7. Month: este campo te permite seleccionar un mes concreto de un año concreto
  8. Number: este campo nos permite introducir numeros y validará correctamente si lo introducido se encuentra entre un maximo y un minimo especificos
  9. Password: este campo permite introducir texto que saldrá en pantalla de forma oculta
  10. Range: este campo nos mostrará un slider que podremos mover para coger un valor entre un minimo y un maximo
  11. Tel: este campo solo permite introducir numeros de telefono
  12. Time: campo para seleccionar la hora
  13. Url: campo para insertar un url
